Heart Disease Hypothesis

The purpose of this project was to use a dataset to help predict Heart Disease. To accomplish this analysis, I first pulled a publicly available dataset from Kaggle.

Visit Heart Disease original dataset site

Visit Kaggle dataset

According to the WHO website https://www.who.int/news-room/fact-sheets/detail/cardiovascular-diseases-(cvds), CVD's are the number 1 cause of death globally: more people die annually from CVDs than from any other cause

To use the data and make a prediction, I used the logistic regression model. "Logistic regression is a classification algorithm. It is used to predict a binary outcome based on a set of interdependent variables." Visit source website
